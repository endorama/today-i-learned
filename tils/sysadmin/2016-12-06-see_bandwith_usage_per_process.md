# See bandwith usage per process
 
_Category: sysadmin_
_Generated on 2016-12-06_

_source: [1]_

```
$ sudo apt-get install nethogs
```

Nethogs groups network usage by process.

Syntax:

```
nethogs [option] [ifaces]
```

Example:

```
$ sudo nethogs eth0
```
